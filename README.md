🎓 SkillSphere – Online Learning Platform

SkillSphere is a modern, dark-themed online learning platform designed to help users explore and enroll in skill-based courses such as Web Development, Design, Marketing, and Data Science. The platform offers a smooth and engaging user experience with an animated hero slider, featured courses, trending sections, and instructor highlights, all wrapped in a visually appealing glassmorphism UI.

Users can browse all available courses with search and filtering options, while detailed course pages are protected and require authentication. The application includes a secure authentication system with email/password login and Google OAuth, along with user profile management features where learners can view and update their personal information.

Built using Next.js, Tailwind CSS, DaisyUI, Prisma, and PostgreSQL, SkillSphere ensures fast performance, clean architecture, and full responsiveness across mobile, tablet, and desktop devices. This project demonstrates a complete full-stack implementation with modern tools, focusing on usability, design, and real-world functionality.
